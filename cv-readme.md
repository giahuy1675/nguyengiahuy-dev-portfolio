# CV Sample File

Để CV viewer hoạt động, bạn cần đặt file CV PDF của bạn vào thư mục public với tên `cv.pdf`

Hoặc bạn có thể:
1. Upload CV lên Google Drive và lấy link public
2. Host trên GitHub
3. Sử dụng bất kỳ hosting service nào

Sau đó cập nhật đường dẫn trong:
- src/components/CVViewer.jsx (dòng 8)
- src/components/Hero.jsx (dòng 14)
- src/components/About.jsx (dòng 11)
